Hide information in plain sight

Want to disguise information as an innocuous image? Stego gives people the ability to hide any file in any image. Once the file is encoded into the image it can be transferred to anyone and they can use Stego to retrieve the file.

Stego does have limitations where the image size must be larger than hidden data, some platforms have limitations on max supported image size for encoding, and images will lose all data if compressed or altered
